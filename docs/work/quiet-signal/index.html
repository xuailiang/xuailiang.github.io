Quiet Signal

一个把个人叙事、屏幕气质与数字产品感折叠在一起的在线入口，轻轻留下技术背景的痕迹，但不把它摆到最前面。

项目背景

我一直想做一个不那么像“技术人员主页”的网站。它不需要用技术栈、时间线和项目数量去证明什么，而是更像一张留下气味的数字名片。

起点

真正让我在意的问题不是“该展示多少信息”，而是“如何让一个页面在很安静的情况下，仍然能让人记住它”。因此我把重点放在氛围、节奏、文字留白，以及画面和交互之间那种不太被察觉的联系。

表达方式

这个页面里当然有代码、有结构，也有产品思维在背后支撑，但它们都没有站到台前。技术在这里更像是一种地基，让视觉和情绪可以站得更稳。

留下来的东西

最终它成为一个既不完全是作品集，也不完全是简历站的入口。它只做一件事：在第一眼里说明我的审美，在第二眼里隐约让人知道，我和数字世界有很深的关系。
